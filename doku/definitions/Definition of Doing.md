# Definition of Doing
Alle Arbeiten ordentlich zusammen, das heißt:
- Der Code wird in Deutsch ausgiebig kommentiert
- Klassen, usw. haben englische Namen
- Unnötige Leerzeilen werden gelöscht, unterschiedliche Klassen etc. sind mit 2 Leerzeilen voneinander zu trennen
- Wenn ein Pull request erstellt wird, dann ist dieser Teil möglichst 100% funktionsfähig und ordentlich auskommentiert
- Pull Requests haben eine Beschreibung
- Teammeatings in DC
